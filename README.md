# CSEC_DATASCIENCE_BOOTCAMP
### 1. String Reverser
This script takes any text you type and flips it backward using Python's slicing technique (`[::-1]`).
- **File:** `reverse.py`
- **Logic:** `res = string[::-1]`

### 2. Vowel Counter
This script counts how many vowels (`a, e, i, o, u`) are in a given sentence. It uses a **Set** for fast lookup and a **Loop** to check each character.
- **File:** `vowel_count.py`
- **Logic:** Checks if each character `i` exists in `{'a', 'e', 'i', 'o', 'u'}`.

### 3. Simple CLI Calculator

A beginner-friendly Command Line Interface (CLI) calculator built with Python. This project demonstrates how to use basic functions, user input, and conditional logic.
